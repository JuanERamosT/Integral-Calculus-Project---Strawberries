# 🍓 Proyecto Final — Cálculo Integral

Modelamiento matemático de una fresa como sólido de revolución, cálculo de volumen mediante integración y diseño de empaques optimizados.

## Estructura

```
proyecto_web/   → Aplicación web Flask (visualización 3D interactiva)
ToSend/         → Informe final en LaTeX (formato IEEE)
```

## Aplicación Web

La aplicación permite visualizar la fresa como sólido de revolución generado por la función:

$$f(x) = -0.3(x - 1)^3 + 2$$

### Ejecución

```bash
cd proyecto_web
pip install -r requirements.txt
python app.py
```

La app estará disponible en `http://localhost:5000`.

## Informe

El informe en formato IEEE se encuentra en `ToSend/informe_proyecto.tex`.
